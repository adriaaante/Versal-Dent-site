# Versal-Dent-site

Сайт премиальной стоматологии **«Версаль-Дент»** — versal-dent.ru.

Плоская статика (HTML/CSS/JS, без сборщика). Бежево-золотой премиум-стиль,
полная SEO-разметка (sitemap, robots, JSON-LD `Dentist`/`FAQPage`/
`BreadcrumbList`, Open Graph), форма заявки → PHP-прокси → Telegram,
Яндекс.Метрика с целями для Директа.

## Структура
- Корневые страницы: главная, о клинике, услуги, врачи, акции, блог,
  отзывы, контакты, юридические (`legal/oferta/privacy/consent/cookies`),
  `404`.
- `services/` — индекс + 9 страниц услуг; `doctors/` — индекс + 5 врачей;
  `blog/` — индекс + 6 статей.
- `assets/` — `css/styles.css` (дизайн-токены в `:root`), `js/`, `img/`.
- `api/lead.php` — приём заявок (конфиг — на сервере, не в репо).
- `scripts/` — деплой и утилиты (фавиконки, водяной знак).

## Документация
- **`CLAUDE.md`** — архитектура, список заглушек к заполнению, деплой,
  workflow.
- **`BRAND.md`** — фирменная палитра и типографика (для сайта и рекламы).

## Локальный просмотр
```
python3 -m http.server 8000
# открыть http://localhost:8000/
```
(Форма заявки требует PHP-бэкенда — работает только на хостинге.)

## Деплой
На сервере reg.ru: `~/deploy.sh` (см. `CLAUDE.md`). Разработка — ветка
`claude/pensive-wright-7oxce`.

> ⚠️ Перед запуском заменить заглушки `[…]` (телефон, адрес, врачи,
> реквизиты, id Метрики и пр.) — полный список в `CLAUDE.md`.
